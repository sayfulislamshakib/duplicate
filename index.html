Duplicate Figma Plugin

Duplicate frames intelligently in any direction with consistent spacing, sibling pushing, and optional blank templates so your design system grows without pushing other layers askew.

Download latest release See how it works

Key Capabilities

Gap Control

Auto-detect spacing from the first frame (40px standard / 100px for wide boards) or type a custom gap when Auto Detect is toggled off.

Push Mode

Siblings that would overlap the duplicate move in the same direction, so no manual nudging is required.

Blank Copies

Create clean starter frames with no fills, strokes, effects, children, or layout constraints for flexible prototyping.

Auto Layout Safe

Auto Layout parents insert clones inline while other sections expand safely to preserve padding around new frames.

How to Use

  1. Select one or more frames or artboards you want to duplicate.
  2. Open Plugins ▸ Development ▸ Duplicate in Figma.
  3. Choose a direction (Right, Left, Top, Bottom, or any diagonal) to place the copy.
  4. Toggle settings for auto gap detection, sibling push, or blank duplicates.
  5. Watch the notification to confirm the gap, Auto Layout handling, and blank clone status.

Settings persist via figma.clientStorage, so your preferred gap and toggles survive between launches.

Settings Overview

Auto Detect Gap

Enabled by default—infers spacing from the selection width. Disable it to manually type a gap value.

Push Siblings

Shifts neighbors along the duplication axis so layouts stay tidy without extra effort.

Duplicate as Blank

Leaves behind structure only—no styles, children, or layout, so you can build fresh content beside the original.

Development Notes

  1. Import via Plugins ▸ Development ▸ Import plugin from manifest... and point to this repo's manifest.json.
  2. Tweak duplication logic, sibling pushing, and notifications inside code.js.
  3. Customize the experience through plain HTML/CSS/JS in ui.html and reload the plugin to preview.
  4. Keep manifest.json in sync when renaming entry files or adding an assets/ folder.
  5. Run the plugin directly in Figma for validation—use figma.notify or console logs for debugging.

Useful Notes

  • figma.clientStorage keeps the last gap, push toggle, and blank toggle saved between runs.
  • Auto Layout parents insert clones inline and skip sibling pushes by design.
  • expandSectionIfNeeded keeps Section nodes padded after you add new frames.
